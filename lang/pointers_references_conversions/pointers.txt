Pointers

&a - адрес ячейки a, в 16тиричном виде, только к lvalue. тип указателя на int -- int*, float -- float*, ...
int a = 1
int* pa = &a;
*pa - операция разыменования, получение значения по адресу; кросме того звездочка может записать в переменную значение *px = 10
std::cout << (*pa) << "\n" // вывод 1 ; std::cout << (*px) << "\n" // выведет 10
